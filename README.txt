This app works for only android mobile phone devices, not tablets, and has been tested and developed on android API 30.
Make sure that the proper permissions are granted on your device in the Android Manifest file:
  <uses-permission android:name="android.permission.INTERNET" />
  <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>
Chord builder only has the first five frets on both instruments, and only common chords have been added for ukulele.
If you run into the problem of tables not being created in databases, delete the database and re run program.
